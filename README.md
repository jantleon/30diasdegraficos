Desde R para Ciencia de Datos han comenzado el reto de compartir un gráfico hecho con R cada día a partir del 12 de mayo. Como el objetivo de esta iniciativa es aprender sobre visualización de datos y ese es uno de mis objetivos actualmente, me voy a sumar al reto. Esta comunidad propone hashtags habitualmente en Twitter para concentrar visualizaciones y fomentar que los usuarios compartan sus gráficos. Aunque la actividad en Twitter de este perfil es muy interesante, su mayor aportación es la traducción del libro R para Ciencia de Datos, del creador de Tidyverse Hadley Wickham. Se puede consultar en el siguiente enlace <https://es.r4ds.hadley.nz/>.
